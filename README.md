📐 Mathematics for Machine Learning — 3-Day Sprint
A focused 3-day self-study program covering the core linear algebra concepts every ML practitioner needs. Rather than a formal course, this is a hands-on math sprint — working through concepts from scratch with NumPy to build real intuition behind the algorithms.
Topics Covered
Vectors

Vectors in Rⁿ — representation and geometry
Vector algebra — addition, subtraction, scaling
Linear combinations of vectors
Linear independence & dependence
Orthonormality — perpendicular unit vectors and why they matter in ML

Matrices

Matrix types — diagonal, triangular, identity
Matrix algebra — addition, multiplication, scalar operations
Transpose of a matrix
Determinant — measuring scale and invertibility
Inverse of a matrix — and the OLS formula β = (XᵀX)⁻¹Xᵀy

Vector Spaces

Foundations of vector spaces — span, basis, and dimension

Why This Matters for ML
Every concept here shows up directly in real ML:

Matrix multiplication powers forward passes in neural networks
Inverse & determinant appear in linear regression and Gaussian models
Orthonormality is the backbone of PCA and SVD
Vector spaces define how your data lives in high-dimensional space
